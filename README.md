# Разработка алгоритма оценки потенциала озеленения зданий

## Актуальность

Научно-исследовательская работа посвящена разработке генеративного алгоритма оценки потенциала размещения зелёных крыш и вертикальных систем озеленения в городах с учетом климатических условий различных регионов России.

## Проблема

В российских городах доля зелёных площадей составляет 8–12% при рекомендуемой норме 40–50%. Существующие подходы к выбору типа озеленения для конкретного здания носят экспертный характер, для чего свойственны: субъективность, низкая воспроизводимость и невозможность масштабирования на город целиком. Национальный проект (2025–2030) ставит задачу увеличения зелёного покрытия на 15–20%, но формализованных инструментов для этого нет.

## Подход

Алгоритм объединяет три метода:

1. **NSGA-II** (генетический алгоритм многокритериальной оптимизации) — генерирует множество Парето-оптимальных конфигураций озеленения, учитывая одновременно 5 критериев
2. **AHP** (метод анализа иерархий) — определяет веса критериев на основе приоритетов пользователя
3. **TOPSIS** — ранжирует решения из Парето-фронта и выдаёт итоговые рекомендации
